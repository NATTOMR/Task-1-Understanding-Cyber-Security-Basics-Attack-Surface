## 1Ô∏è‚É£ What is Cyber Security?
Cybersecurity is the practice of protecting systems, networks, and data from digital attacks.

### CIA Triad
- **Confidentiality** ‚Äì Prevents unauthorized access  (e.g., bank account details).
- **Integrity** ‚Äì Prevents unauthorized data modification  (e.g., transaction records).
- **Availability** ‚Äì Ensures services are accessible  (e.g., online banking services).
